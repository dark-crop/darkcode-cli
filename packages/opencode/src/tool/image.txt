Generate an image from a text prompt, or edit an existing image, using the Dark-LLM gateway. Use this whenever the user asks to create, generate, draw, make, or edit an image / picture / icon / logo / diagram / mockup.

Two modes:
- GENERATE (text-to-image): provide `prompt` describing the image. Set `size` ("WxH" pixels) to control resolution / aspect ratio - convert any ratio the user asks for (16:9, portrait, square, etc.) into pixels near ~1MP per the `size` param guide. Backed by z-image (~24s). This is the default.
- EDIT (image-to-image): set `mode` to "edit" and put the change to make in `prompt`. **If the user attached / pasted / gave an image in their message, it is used automatically as the input - you do NOT need any file path.** Only pass `images` (1-3 absolute paths) to edit specific files on disk instead. Backed by qwen-image-edit (~2.5 min).

IMPORTANT: if the user gives/attaches an image and asks to change, modify, edit, add to, remove from, restyle, or "make this ..." it, that is EDIT - call the tool with mode:"edit" (do NOT generate a new image from scratch). The result is saved as a PNG in the workspace and the tool returns the path. Actually call this tool - do not describe an imaginary image.
